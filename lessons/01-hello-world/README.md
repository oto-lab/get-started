# Lesson 01 — Hello, World

[日本語](./README-ja.md)

Welcome to Purus! Let's start with the classic first program.

## Syntax

Purus strings are written between `//;` and `;//`:

```purus
//;Hello;//
```

Function calls use `[]` instead of `()`:

```purus
console.log[//;Hello;//]
```

## Task

Edit `main.purus` so it prints:

```
Hello, Purus!
```

## Check

```sh
npm test 01
```
