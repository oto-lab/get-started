# Lesson 01 — Hello, World

[English](./README.md)

Purus へようこそ！まずは定番の最初のプログラムから始めましょう。

## 構文

Purus の文字列は `//;` と `;//` で囲んで書きます:

```purus
//;Hello;//
```

関数呼び出しは `()` ではなく `[]` を使います:

```purus
console.log[//;Hello;//]
```

## 課題

`main.purus` を編集して、次のように出力してください:

```
Hello, Purus!
```
