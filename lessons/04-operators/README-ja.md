# Lesson 04 — 演算子

[English](./README.md)

## 構文

Purus は記号の演算子をキーワードに置き換えているので、Shift キーが不要です:

| Purus | JavaScript | 意味 |
|---|---|---|
| `a add b` | `a + b` | 加算 |
| `a sub b` | `a - b` | 減算 |
| `a mul b` | `a * b` | 乗算 |
| `a div b` | `a / b` | 除算 |
| `a mod b` | `a % b` | 剰余 |
| `a pow b` | `a ** b` | べき乗 |

## 課題

`a be 7`、`b be 3` として、上の演算子を使って `sum`・`difference`・`product`・`remainder`・`power` を計算してください。
